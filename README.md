# NsTut Lib

A library mod for NsTut's mods.

## Overview

NsTut Lib is a Minecraft modding library used by other mods created by NsTut. It provides common utilities and functionalities to streamline development and ensure consistency across NsTut's mod portfolio.

## Installation

This library is intended to be used as a dependency for other Minecraft mods. If you are a developer, you can include it in your project by adding it to your `build.gradle` file.

## For Users

If you are a player, you typically don't need to download this mod directly. It will be automatically downloaded if required by another mod from NsTut that you are using.

## Building from Source

1. Clone the repository: `git clone https://github.com/your-username/NsTut-Lib.git` (Replace with your actual repository URL)
2. Navigate to the project directory: `cd NsTut-Lib`
3. Build the mod using Gradle: `./gradlew build`

The compiled mod files for each platform will be located in their respective subproject `build/libs` directories (e.g., `fabric/build/libs` and `forge/build/libs`).
